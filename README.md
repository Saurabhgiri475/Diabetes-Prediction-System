# 🩺 Diabetes Prediction System

A Machine Learning web application built with **Python**, **Scikit-learn**, and **Streamlit** to predict whether a patient is likely to have diabetes based on various health parameters.

## 🚀 Features

- Predicts diabetes using a trained Random Forest Classifier
- User-friendly Streamlit interface
- Manual encoding of categorical features
- Instant prediction results
- Simple and lightweight application
- Easy to deploy on Streamlit Community Cloud

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Diabetes-Prediction-System.git
```

Move to the project folder:

```bash
cd Diabetes-Prediction-System
```

Install the required packages:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
[🚀 Launch Diabetes Prediction App](https://diabetes-prediction-system-saurabh12.streamlit.app/)
```

The application will open automatically in your browser.

---

## 🧠 Machine Learning Model

- Algorithm: Random Forest Classifier
- Task: Binary Classification
- Target Variable: Diabetes

---

## 📋 Prediction Inputs

| Feature | Description |
|---------|-------------|
| Age | Patient Age |
| Gender | Male/Female/Unknown |
| BMI | Body Mass Index |
| Glucose | Blood Glucose Level |
| Blood Pressure | Blood Pressure |
| Cholesterol | Cholesterol Level |
| Insulin | Insulin Level |
| HbA1c | Average Blood Sugar |
| Smoking Status | Current/Never/Former |
| Physical Activity | Low/Moderate/High |
| Family History | Yes/No |

---

## 📈 Output

The application predicts whether the patient is:

- ✅ No Diabetes Detected
- ⚠️ High Risk of Diabetes
